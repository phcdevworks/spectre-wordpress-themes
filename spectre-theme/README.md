# Spectre WordPress Themes Theme Directory

This directory contains the default WordPress theme files for Spectre WordPress Themes.

## Development notes

1. Keep theme metadata in style.css aligned with the repository identity.
2. Keep PHP functions and translation domains consistent with the default theme prefix.
3. Update templates in this directory as the WordPress theme surface evolves.
4. Run npm run dev in the project root during development.
5. Run npm run build before deploying.

## Deployment

Deploy the full spectre-theme directory to your WordPress installation:

    wp-content/themes/spectre-theme/

The dist directory is generated by Vite and contains the compiled theme assets.

See the main README.md in the project root for full project documentation.
